# MaaNTE 问题排查手册

## 运行问题

### 无法启动

#### 弹窗提示To run this application, you must install .NET

- 前往[下载 .NET 10.0](https://dotnet.microsoft.com/zh-cn/download/dotnet/10.0) 并安装 `.NET 10.0 Desktop Runtime` （.NET 桌面运行时）。

### 无法连接窗口

1. 确保你打开了异环。
2. 确保你是**以管理员身份运行**的 MaaNTE 。
3. 将异环设为`1280x720`分辨率，窗口化。

## 任务问题

### 通用

#### 我的功能比别人少

- 部分功能需要**选择特定控制器**才会出现，不妨试试切换控制器
- 目前主页的切换控制器功能存在问题，请在`设置-连接设置`里更改。前台控制器为鼠标`Seize`，后台控制器为鼠标`SendMessageWithWindowPos`

#### 一开始就立刻完成

- 检查左侧`任务列表`中有没有勾选相应任务。
- 关闭游戏内插帧、超分辨率等影响画质的功能。

#### 无法连接窗口/开始任务

- 确保你将 MaaNTE 添加到了杀毒软件的白名单里，或关闭了杀毒软件。
- 确保游戏语言为**简体中文**。

#### 无法正常进行点击/操作

- 确保你的 MaaNTE 处于**纯英文**路径下，并且没有全角符号（最好符号也别有）。
- 确保你是**以管理员身份运行**的 MaaNTE 。
- 无法正常进行点击时尝试将鼠标输入方式改为`Seize`
- 确保 Windows 屏幕缩放为 100%

#### 抢占鼠标问题

- 在`设置-连接设置`里将鼠标改为`SendMessageWithWindowPos`即可，但是部分标明需要前台控制器的任务需要`Seize`（会抢鼠标）。

#### 识别失败

- 关闭游戏内插帧、超分辨率等影响画质的功能

- 见[任务问题-通用-无法连接窗口/开始任务](#无法连接窗口开始任务)

- 见[任务问题-通用-无法正常进行点击/操作](#无法正常进行点击操作)

- 见[任务问题-通用-无法正常进行点击/操作](#无法正常进行点击操作)

#### 无法售卖渔获

- 尝试将游戏改为120帧。
- 内测版正在测试新的方案，敬请期待。

#### 无法购买鱼饵

- 降低`鱼饵识别阈值`

#### 钓不到鱼

- 内测版正在测试新的方案，敬请期待。

#### 钓鱼任务提前结束

- 保证你剩余鱼饵多余每轮钓鱼数
